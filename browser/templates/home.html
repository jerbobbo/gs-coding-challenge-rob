GlamSquad Coding Challenge

Feature One - New Server Side Routes
  • add a server side route which returns a single category along with all of it's services.
    GET: /api/categories/:categoryId
  • add a server side route which deletes a service from a category
    DELETE: /api/categories/:categoryId/services/:id
  • add a server side route which adds a service to a category
    POST: /api/categories/:categoryId/services
    EXAMPLE PAYLOAD: { name: 'a new service' }
Feature Two - Show Category Details
Create front end code which shows the category details when a category link is clicked.
Feature Three - Create a Service
Enable the user to creater a new service on the category detail page
Feature Four - Delete a Service
Enable the user to delete a service on the category detail page.
